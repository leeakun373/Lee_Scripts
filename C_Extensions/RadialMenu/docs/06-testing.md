# 手动测试清单

## 基础打开/关闭

1. 绑定 `Lee_RadialMenu_Open` 热键：按下出现、松开消失（未 Pin）
2. 按住超过 2 秒仍稳定
3. ESC：未 Pin 关闭；Pin 时只收起子菜单

## 执行

4. 点击 action：按住热键期间不重复弹出
5. Pin 下执行不意外关闭
6. FX/chain：选中 item / track / 无选中新建轨
7. template：打开工程模板

## 拖拽

8. Setup：Browser → Slot
9. 运行时：Slot → Arrange（item / track / 空白）

## 热重载

10. 打开运行时轮盘 → Setup 保存 → 样式/尺寸自动刷新

## 管理模式

11. 内圆右键进入/退出；切换预设无窗口漂移

## Setup

12. 新建/复制/重命名预设；Default 不可删
13. 语言 zh/en 切换
14. Action/FX 长列表滚动流畅

## DPI

15. 125%/150% 缩放 + 双显示器：轮盘锚点不漂移
